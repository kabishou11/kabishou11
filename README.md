### 🤖 DocForge
> 基于大语言模型的智能文档生成工具

- 输入主题和描述即可自动生成专业文档
- 支持参考现有文档风格生成格式统一的新内容
- 交互式终端界面 (TUI)，支持斜杠命令
- 支持 Markdown 和 DOCX 导出

**Tech Stack:** TypeScript · Node.js · Python · ModelScope API · Qwen3 · PaddleOCR
